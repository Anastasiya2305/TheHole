# The Hole

Prototype game in Unity where the player controls a hole that consumes fruits and fruit pieces.

## Core Idea
The player moves a hole around the level and collects edible objects.  
The project is focused on building a clean, expandable gameplay prototype with data-driven systems.

## Planned Mechanics
- normal fruits
- shelled fruits with states:
  - Closed
  - Cracked
  - Opened
- spawning fruit pieces after opening
- overripe fruits
- reduced score for spoiled fruits
- sticky puddles
- hole slowdown inside sticky zones

## Tech Stack
- Unity
- C#
- Git
- GitHub

## Project Structure
- `Assets` — scenes, scripts, prefabs, materials
- `Packages` — Unity packages
- `ProjectSettings` — Unity project settings

## Current Status
Initial Unity project created and connected to Git.

## Development Goals
